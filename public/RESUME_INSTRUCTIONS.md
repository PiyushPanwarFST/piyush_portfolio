# Resume Setup Instructions

## How to Add Your Resume

1. **Get your resume PDF ready**
   - Make sure your resume is in PDF format
   - Name it `resume.pdf`

2. **Add to the project**
   - Place your `resume.pdf` file in the `/public` folder
   - The file should be at: `/public/resume.pdf`

3. **Verify**
   - The download button in the Hero section will automatically work
   - The file will be downloadable as "Rishav_Tarway_Resume.pdf"
